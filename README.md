```
A CLI tool to automate Zimbra Web Client tasks

Usage:
  zwc [command]

Available Commands:
  draft       Save/Edit/Delete a draft mail message
  help        Help about any command

Flags:
      --config string   config file (default is $HOME/.zwc.yaml)
  -h, --help            help for zwc

Use "zwc [command] --help" for more information about a command.
```
